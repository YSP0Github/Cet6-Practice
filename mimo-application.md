# 04 项目描述

我使用 Claude Code 独立开发并上线了一个 **大学英语六级（CET-6）真题在线练习系统**。项目已开源至 GitHub：<https://github.com/YSP0Github/Cet6-Practice>，并部署在：<https://cet6.yshome.top>。

## 项目背景与核心痛点

在备考大学英语六级的过程中，真题、解析和听力音频通常分散存放在不同文件夹或资料包中，学生需要反复查找、下载和切换应用，学习流程被频繁打断。尤其在手机端，浏览器往往无法稳定地同时展示真题 PDF、解析 PDF 并播放听力音频，导致移动学习体验较差。

针对这些问题，本项目将 **2015—2025 年共 23 套六级真题资源**（真题 PDF、解析 PDF、听力音频）整合为一个纯前端在线练习平台。用户无需安装任何 App，即可在同一页面内完成“听听力、看真题、查解析”的完整练习流程。

## 核心功能与技术实现

项目采用纯前端静态站点方案，无后端依赖，部署和维护成本较低。用户可以在首页通过搜索或浏览快速定位目标套题，并进入练习页面：桌面端采用左右分栏布局，左侧展示真题 PDF，右侧展示解析 PDF，顶部集成听力播放器，便于边听边做题、随时对照解析。

针对移动端浏览器对内嵌 PDF 支持不稳定的问题，我在 Claude Code 的辅助下实现了基于 **PDF.js** 的移动端适配方案：系统会在手机端自动切换为标签页布局，将“听力 / 真题 / 解析”拆分为三个 Tab，并使用 PDF.js 将 PDF 逐页渲染到 Canvas 中，从而保证移动端也能获得完整、流畅的练习体验。

除核心练习流程外，系统还实现了资源搜索、收藏管理、最近访问记录、学习笔记、每日计划等辅助功能，并使用 `localStorage` 在本地持久化用户数据，帮助学习者更好地管理复习进度。

## AI 辅助开发价值

在开发过程中，我主要借助 Claude Code 完成需求拆解、代码实现、响应式适配、移动端 PDF 渲染方案设计、代码审查与部署配置优化。整个项目经过多轮迭代，累计优化约 2000 行前端代码。通过这个项目，我不仅将分散的六级备考资料整理成可直接使用的在线工具，也验证了 AI 编程助手在真实项目开发、问题定位和体验优化中的实际价值。
