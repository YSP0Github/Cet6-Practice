# 大学英语四六级真题练习系统

一个纯本地运行的真题练习页面，把四六级真题、解析与听力统一到一个界面，方便集中复习与记录错题。支持 CET-4（四级）和 CET-6（六级）切换。

## 主要特性

- **四六级切换**：通过顶部标签一键切换四级/六级，数据、笔记、收藏互相独立。
- **资源库**：按年份与套题罗列真题 PDF、解析及听力音频，点击即可打开。
- **听力播放**：内置播放器直接播放本地 MP3，跟随套题自动切换。
- **最近访问与收藏**：快速回到常用资料，减少搜索时间。
- **错题笔记**：本地保存练习体会与计划，刷新也不会丢失。
- **每日计划**：简单打卡，保持学习节奏。
- **沉浸模式**：隐藏其它模块，只保留真题与解析分屏阅读。
- **移动端适配**：手机端使用 PDF.js 渲染 PDF，支持标签页切换浏览。

## 资源覆盖

| 级别 | 年份范围 | 考试期数 |
|------|---------|---------|
| CET-4（四级） | 2021-2025 | 10 期 |
| CET-6（六级） | 2015-2025 | 25 期 |

## 使用方式

1. 在浏览器中打开 `index.html`。
2. 推荐使用 Edge 或 Firefox 访问本地文件，如受限可开启本地静态服务器。
3. 通过顶部的”四级/六级”标签选择目标级别。
4. 在”最新套题”或”真题资源库”中点击需要的套题，即可打开预览或直接播放音频。
5. 需要快速全屏体验时，点击”沉浸模式”进入分屏阅读与听力播放界面。
6. 在”错题笔记”区记录内容，点击”保存笔记”即可存储在本地浏览器。

## 扩展数据

如需增加新的真题资料，可将 PDF 和音频文件放入对应目录，然后运行 `generate_manifest.py` 重新生成数据文件：

- 六级文件：`pdf/` 和 `audio/`
- 四级文件：`pdf/cet4/` 和 `audio/cet4/`

文件命名规范：`YYYYMM_N.pdf`（真题）、`YYYYMM_N_解析.pdf`（解析）、`YYYYMM_N.mp3`（听力）。

## 练习建议

- 先独立完成真题，再对照解析复盘错题，避免”看答案就会”的错觉。
- 听力建议三遍：整体理解、细节补充、跟读复述。
- 记录高频词汇、长难句与写作亮点，形成个人资料库。
- 听力、阅读、写作轮换练习，保持每周至少两套题的节奏。
