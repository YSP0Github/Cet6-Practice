# 大学英语六级真题练习系统

一个纯本地运行的真题练习页面，把你收藏的六级真题、解析与听力统一到一个界面，方便集中复习与记录错题。

## 主要特性

- **资源库**：按年份与套题罗列真题 PDF、解析及听力音频，点击即可打开。
- **听力播放**：内置播放器直接播放本地 MP3，跟随套题自动切换。
- **最近访问与收藏**：快速回到常用资料，减少搜索时间。
- **错题笔记**：本地保存练习体会与计划，刷新也不会丢失。
- **每日计划**：简单打卡，保持学习节奏。
- **沉浸模式**：隐藏其它模块，只保留真题与解析分屏阅读。

## 使用方式

1. 在浏览器中打开 `六级真题练习系统/index.html`。
2. 推荐使用 Edge 或 Firefox 访问本地文件，如受限可开启本地静态服务器。
3. 在“最新套题”或“真题资源库”中点击需要的套题，即可打开预览或直接播放音频。
4. 需要快速全屏体验时，点击“沉浸模式”进入分屏阅读与听力播放界面。
5. 在“错题笔记”区记录内容，点击“保存笔记”即可存储在本地浏览器。

## 扩展数据

如需增加新的真题资料，可编辑 `app.js` 中的 `libraryData` 数组，按照现有结构追加年份、标签与资源路径。

## 练习建议

- 先独立完成真题，再对照解析复盘错题，避免“看答案就会”的错觉。
- 听力建议三遍：整体理解、细节补充、跟读复述。
- 记录高频词汇、长难句与写作亮点，形成个人资料库。
- 听力、阅读、写作轮换练习，保持每周至少两套题的节奏。
